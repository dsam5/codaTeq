1. Python programming, mathematics, and statistics for data science

This foundational course is designed for beginners with no prior experience in programming or data science. It focuses on Python programming and essential mathematical and statistical concepts required to excel in data science. You’ll learn how to code in Python, handle data structures, and apply statistics and mathematics to analyze data.

What You'll Learn

  • Fundamental Python programming skills (data types, loops, functions, libraries like NumPy and Pandas)
  • Key mathematical concepts such as linear algebra and calculus
  • Essential statistics for data science, including probability, distributions, and hypothesis testing
  • Data manipulation and exploratory data analysis (EDA)

Requirements

  • No prior programming experience required
  • A laptop and stable internet connection
  • Willingness to learn and dedicate time to practice

2. Machine learning with Python

This course takes you from the foundational skills of Python programming to applying machine learning models on real-world data. You'll learn how to build and deploy predictive models, covering both supervised and unsupervised techniques. This course is designed for those with basic Python knowledge looking to delve into machine learning.

What You'll Learn

  • Core machine learning algorithms: Linear Regression, Logistic Regression, Decision Trees, Random Forests, and more
  • Supervised and unsupervised learning techniques
  • Feature engineering and model evaluation techniques
  • Hyperparameter tuning, cross-validation, and model optimization
  • Introduction to time series models for forecasting
  • Hands-on projects applying machine learning models to real-world datasets

Requirements

  • Completion of "Python Programming, Mathematics, and Statistics for Data Science" or prior knowledge of Python and basic statistics
  • Familiarity with Python libraries such as NumPy, Pandas, and scikit-learn
  • Enthusiasm for data science and a commitment to learning.
  • A laptop and stable internet connection

3. Deep Learning

Dive deep into the world of neural networks and advanced AI techniques with this course. You'll learn the fundamentals of deep learning and how to build, train, and deploy neural networks using frameworks like TensorFlow and PyTorch. This course is perfect for those with a background in machine learning looking to expand their knowledge into neural networks.

What You'll Learn

  • Fundamentals of deep learning and artificial neural networks (ANN)
  • Recurrent Neural Networks (RNN) and Long Short-Term Memory (LSTM) networks
  • How to build deep learning models using Python (TensorFlow/Keras)
  • Model optimization techniques (dropout, batch normalization, and hyperparameter tuning)
  • Hands-on deep learning projects with real-world applications

Requirements

  • Completion of the "Machine Learning with Python" course or prior experience in machine learning concepts
  • A solid understanding of Python programming and basic linear algebra
  • Familiarity with Python libraries such as NumPy and scikit-learn
  • A laptop and stable internet connection
